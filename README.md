# Study Buddy — Time Manager

A minimal Python-based Study Planner that generates a personalized study schedule based on your available time and subject priorities.  
Designed as a clean beginner → intermediate CLI project.

---

## Features
- Generates a structured study plan
- Distributes time based on subject priority
- Suggests break intervals
- Shows a motivational quote loaded from a JSON file
- Simple, extendable Python architecture
